# anime2watch

This is a anime library project to showcase the usage of the following stack

- SSR: Nextjs
- Styles: Tailwind/Shadcn
- Animations: Framer Motion
- SEO: Nextjs Meta Tags
- Login: Clerk
- API : Apollo GraphQL
